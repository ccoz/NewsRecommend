# 基于协同过滤算法的新闻推荐系统
# 技术栈
1. 后端使用SpringBoot+Mybatis-Plus框架
2. 前端使用Vue.js+Element-Plus+Windi CSS框架
3. 数据爬虫使用Python+Scrapy框架
4. 大数据推荐功能使用Scala+Spark实现协同过滤算法，采用余弦相似度计算。
5. 使用Docker+Maven打包部署服务器
